# ☀️ YWS Solar - Hero Redesign

> Projeto experimental para desenvolver a nova tela inicial (Hero) da plataforma YWS Solar.

## 🎯 Objetivo

Este repositório existe exclusivamente para criar uma nova Hero para a YWS Solar.

Nenhuma alteração deve ser feita nas demais páginas até que a nova Hero seja aprovada.

O objetivo é criar uma landing page moderna, profissional e otimizada para conversão.

---

# 📸 Referência Visual

A referência principal encontra-se em:

docs/hero-reference.png

Essa imagem representa a direção visual desejada.

Ela deve servir como inspiração para layout, posicionamento dos elementos e experiência do usuário.

Não é necessário copiar exatamente cada detalhe, mas manter a mesma proposta visual.

---

# 🎨 Conceito

A Hero deve transmitir:

- tecnologia
- confiança
- inovação
- energia limpa
- modernidade
- profissionalismo

Inspirada em empresas como:

- Apple
- Stripe
- Vercel
- Linear
- Framer

---

# 👨 Mascote Yuri

O Yuri será o principal elemento visual da Hero.

Requisitos:

- personagem em alta qualidade
- PNG com fundo transparente
- posição no lado esquerdo
- aproximadamente 35% a 40% da largura da tela
- segurando um celular
- expressão simpática e profissional

O Yuri representa a identidade da YWS Solar.

---

# 📱 Celular

O celular deverá:

- mostrar o site da YWS Solar
- possuir leve inclinação
- possuir sombra moderna
- ficar parcialmente sobreposto ao Yuri

O celular deve chamar atenção para o simulador.

# ✅ Objetivo Final

Criar uma Hero que faça a YWS Solar parecer uma empresa de tecnologia moderna, mantendo a identidade visual da marca e utilizando o Yuri como principal elemento de comunicação.

Este repositório serve apenas para experimentação antes da implementação no projeto oficial.
